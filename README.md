**Sizzle Share** is a cross-platform mobile application designed to simplify the culinary experience. It bridges the gap between recipe discovery, social interaction, and practical meal planning.

Built as a graduation project, this application leverages a full-stack architecture to provide a seamless user experience for food enthusiasts to share their creations and organize their weekly nutrition.
